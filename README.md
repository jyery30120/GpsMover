# iPhone GPS 模擬器

在 MacBook 上透過瀏覽器點擊地圖，模擬 iPhone 以走路／慢跑／騎車速度移動到目的地。

支援 iOS 17+（需要 tunneld）。

---

## 畫面

地圖左側為全螢幕地圖，右側為控制面板。

- 綠點：起點
- 紅點：終點
- 藍點：目前模擬位置

---

## 需求

- macOS
- Python 3.10+
- iPhone 透過 USB 連接並信任此電腦
- iPhone 已開啟**開發者模式**（設定 → 隱私權與安全性 → 開發者模式）

---

## 安裝

```bash
pip install pymobiledevice3 flask
```

---

## 啟動

### 步驟一：啟動 tunneld（只需要做一次，iOS 17+ 必要）

開一個終端機視窗，執行以下指令並保持開著：

```bash
sudo python3 -m pymobiledevice3 remote tunneld
```

> 這個指令需要 sudo 因為它會建立一個 utun 網路介面。只要這個終端機視窗開著，後續就不用再重複執行。

### 步驟二：啟動 Web 介面

```bash
python3 app.py
```

開啟瀏覽器前往 [http://localhost:5050](http://localhost:5050)。

---

## 使用方式

1. 點擊右側面板的**「連接 iPhone」**
2. **第一次點擊地圖**：設定起點（綠點）
3. **第二次點擊地圖**：設定終點（紅點），立即開始模擬移動
4. **移動中點擊地圖**：即時更換目的地（從目前位置重新導航）
5. 點擊**「停止移動」**：停止模擬，iPhone 回復真實 GPS

### 自訂起點

座標面板的起點旁有**「重設」**按鈕：

- 點擊後按鈕閃綠、地圖 cursor 變十字
- 點擊地圖任意位置設定新起點
- 若正在移動中，會先停止再等待設定新終點

### 速度設定

| 預設  | 速度          |
|-------|---------------|
| 🚶 走路 | 1.4 m/s（5 km/h）  |
| 🏃 慢跑 | 2.5 m/s（9 km/h）  |
| 🚲 騎車 | 4.5 m/s（16 km/h） |

也可以拖曳滑桿自訂任意速度（0.5 ～ 5 m/s）。

## CLI 版本（不需要 Web 介面）

```bash
python3 gps.py <起點緯度> <起點經度> <終點緯度> <終點經度> [--speed 速度]
```

範例：

```bash
# 台北101 走到信義誠品（預設走路速度）
python3 gps.py 25.033964 121.564468 25.036533 121.567825

# 慢跑速度
python3 gps.py 25.033964 121.564468 25.036533 121.567825 --speed 2.5
```

按 `Ctrl+C` 停止並清除模擬位置。

## 常見問題

**連接失敗：tunneld 未執行**

先確認步驟一的 `sudo tunneld` 有在執行。

**連接失敗：找不到裝置**

- 確認 iPhone 用 USB 連接
- iPhone 螢幕上確認「是否信任此電腦」→ 點信任
- 確認 iPhone 已開啟開發者模式

**GPS 沒有變化**

- 確認開發者模式已開啟
- 重新拔插 USB，再次點連接
